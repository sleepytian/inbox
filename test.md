# 第六章、应用层

## 6.1、网络应用模型

### 应用层概述

应用层对应用程序之间的通信提供服务。

**应用层协议定义**：

- 应用进程交换报文类型，是请求还是响应？
- 各种报文类型语法，如报文中的各个字段及其详细描述。
- 字段的语义
- 进程何时、如何发送报文，以及对报文进行响应的规则。

**应用层的功能**：

- 文件传输
- 电子邮件
- 虚拟终端
- 查询服务和远程作业登录

**应用层的协议**：

- FTP
- SMTP、POP3
- HTTP
- DNS
